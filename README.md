# 📸 Instagram AI Studio

An AI-powered Instagram Content Creation Platform built with Python, Streamlit, Google Gemini AI, and YOLOv8.

## 🚀 Features

- 🤖 AI Caption Generator
- ✨ Caption Improver
- 📅 Content Calendar Generator
- 🎬 AI Reel Script Generator
- 📈 Competitor Analyzer
- 📊 Profile Auditor
- 📧 Brand Collaboration Email Generator
- #️⃣ AI Hashtag Generator
- 💬 AI Reply Generator
- 😊 Sentiment Analyzer
- 📈 Engagement Predictor
- 🖼️ YOLO Image Analyzer
- 📊 Analytics Dashboard
- 📜 History Management
- 🔐 Login System

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini API
- YOLOv8
- SQLite
- Pandas
- OpenCV
