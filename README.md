# 📸 Instagram AI Studio

An AI-powered Instagram content creation platform built with Python, Streamlit, YOLOv8, and Google Gemini AI.

## 🚀 Features

- 🤖 AI Caption Generator
- ✨ Caption Improver
- 📅 Content Calendar Generator
- 🎬 Reel Script Generator
- 📈 Competitor Analyzer
- 👤 Profile Auditor
- 📧 Brand Collaboration Email Generator
- #️⃣ Hashtag Generator
- 💬 AI Reply Generator
- 😊 Sentiment Analyzer
- 📈 Engagement Predictor
- 🖼️ YOLO Image Analyzer
- 📊 Analytics Dashboard
- 🔐 Login System
- 📜 History Management

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini API
- YOLOv8
- Pandas
- SQLite
- OpenCV

## ▶️ Installation

```bash
git clone https://github.com/aamirsalam97-design/Instagram-AI-Studio.git

cd Instagram-AI-Studio

pip install -r requirements.txt

streamlit run app.py
```
